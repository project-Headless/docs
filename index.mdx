## Requirements

To start using Knot.dev, you will need the following:

- A Knot.dev account
- A Shopify store - don't have one? <a href="https://www.shopify.com/?ref=knotdev" target="_blank" rel="noopener noreferrer">Sign up for a free trial</a>
- An API key and Shopify domain

<Steps>
    <Step title="Install Headless App">
        Install the Headless app from the <a href="https://apps.shopify.com/headless" target="_blank" rel="noopener noreferrer">Shopify App Store</a>.
        - Navigate to the 'Apps' section.
        - Type 'headless' in the search bar.
        - Click on 'Install' to add the headless application.
    </Step>
    <Step title="Create a storefront">
        Once installed, click on the Headless app in your Shopify admin, locate and click on 'Create Storefront'.
    </Step>
    <Step title="Copy your credentials">
        - Click on 'Manage' next to the Storefront API option.
        - Open a note-taking application to store your tokens.
        - Copy the 'Public Access Token' & 'Private Access Token' from the Storefront API settings.
    </Step>
    <Step title="Note your store domain">
        Your Shopify domain is your store's URL in the format `your-store-name.myshopify.com`. You'll need this along with your API token.

        If you forgot your store domain, you can find it in the Shopify admin under "Settings" > "General".

    </Step>
</Steps>
