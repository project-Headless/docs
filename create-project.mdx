## Create a new project

Creating a new project is as easy as it gets with Knot.
You can either use any prompt or image reference you like for Knot Studio,
and for Knot Instant, you can simply click the "Create Instant Project" button to get started.


## Studio vs Instant: Which is right for you?

Compare the key differences to make the best choice for your business

| Aspect | Knot Studio | Knot Instant |
|--------|-------------|--------------|
| **Who** | "I need my store to reflect my brand perfectly" | "I need to start selling quickly" |
| **When** | Building for growth, serving clients, established brand | Testing ideas, first store, simple needs |
| **What** | Custom layouts, unique interactions, full control | Quick launch, essential customizations, proven design |
| **Outcome** | Unique, scalable store that stands out | Professional store ready to sell immediately |
| **Price** | Credits consumed as you go | 10 Credits |
